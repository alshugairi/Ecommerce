### Ecommerce Starter
Starter project created with laravel 11.

## Requirements
- PHP >= 8.2.

## Steps

- clone it
- composer install
- add .env file & change database
- php artisan key:generate
- sudo chmod 777 -R storage
- php artisan migrate
- php artisan db:seed
- php artisan storage:link
- php artisan serve
